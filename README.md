# RadBio - softwarový nástroj pro odhad kontaminace rostlinné  biomasy na území zasaženém jadernou havárií

*Po radiační havárii mohou být kontaminovány radionuklidy (RN) rozsáhlé plochy určené k zemědělské produkci, a tím velmi omezeny možnosti využívání krajiny. Obnovení zemědělské činnosti na zasaženém území bude jednou z priorit. K tomu by měl pomoci i vyvinutý software (SW) RadBio, umožňující odhadnout kontaminaci biomasy rostlin na základě informací o půdních charakteristikách a o stupni kontaminaci půdy.*
